**Read a Little** adds a "Read a Little" button to WooCommerce product pages, allowing customers to preview documents before purchasing.

== Description ==

**Read a Little for WooCommerce** enhances your WooCommerce store by adding a 'Read a Little' button to product pages, specifically designed for books and file-based content. This feature allows your customers to preview PDFs, Word documents, Excel files, and PowerPoint presentations directly on the product page using Google Docs Viewer.

== Key Features ==
- Display previews for PDF files, Excel files, PowerPoint presentations, and Word documents.
- Add the 'Read a Little' button to showcase book previews or related content.
- Use a simple shortcode `[read_little_button]` to insert the preview button anywhere on your product pages.
- Seamless integration with WooCommerce.
- Easily customizable button and file preview styling from the plugin settings.

== Installation ==

= Plugin Installation Method: =

1. Go to the WordPress dashboard.
2. Navigate to **Plugins > Add New**.
3. Search for **"Read a Little for WooCommerce"**.
4. Click the **Install** button.
5. After installation, click **Activate** to activate the plugin.

= Installation via Zip file: =

1. Download the **Read a Little for WooCommerce** plugin zip file.
2. Go to **Dashboard > Plugins > Add New > Upload Plugins**.
3. Select the `read-a-little.zip` file and proceed to install.
4. Activate the plugin.

= Plugin Configuration: =

1. Go to the WooCommerce product edit screen to add file previews for your products.
2. Navigate to the plugin settings page to customize button position, file types, and effects.

== Frequently Asked Questions ==

= Can I customize the button style? =

- Yes, the plugin allows you to customize button styles and positioning through the WordPress admin settings.

= Will this plugin work with my theme? =

- The plugin is designed to work with most WooCommerce-compatible themes. Custom themes may require additional styling adjustments.

= Can I use a shortcode to place the "Read a Little" button? =

- Yes, you can use the `[read_little_button]` shortcode to display the button. If no parameters are provided, the button will use the default files linked to the product.
